WebSocket
WebSocket 是一种网络通信协议。

WebSocket 是 HTML5 开始提供的一种在单个 TCP 连接上进行全双工(full-duplex)通讯的协议。
没有了 Request 和 Response 的概念，两者地位完全平等，连接一旦建立，
就建立了真•持久性连接，双方可以随时向对方发送数据。

WebSocket的出现历程
短轮询效率低，非常浪费资源（网络带宽和计算资源）。有一定延迟、服务器压力较大，并且大部分是无效请求。

长轮询虽然省去了大量无效请求，减少了服务器压力和一定的网络带宽的占用，但是还是需要保持大量的连接。

最后到了基于流的方式，在服务器往客户端推送，这个方向的流实时性比较好。
但是依旧是单向的，客户端请求服务器依然还需要一次 HTTP 请求。
